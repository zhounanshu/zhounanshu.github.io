Sublime定制

Jun 16, 2016 | 技术 | Hits

 Sublime原生的markdown编辑插件MarkDown Editing并不友好。而Monokai extended插件支持对MarkDown进行语法高亮编辑，其安装方法如下： 以下均在Mac OX系统下操作：

  1. 安装 Monokai extended插件 cmd + shift + p调出control panel输入install package.然后 搜索Monokai extended安装插件
  2. 激活插件，并设置 cmd + shift + p，输入set syntax:markdown extended，然后在导航栏view --> Syntax --> Open all with current extension as… -> Markdown Extended
